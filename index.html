Gated control

Allows access to control in a two step process. Can be used to conceal codes or controls as demonstrated in the examples below

Includes
<script src="/path_to_gatedcontrol.js></script>
<link href="/path_to_gatedcontrol.css" rel="stylesheet">
Simple Usage
$(document).ready(function(){
    $('input[type=text]').gatedControl();
});	    				
<input type="text">
Options
$(document).ready(function(){
    $('#example2').gatedControl({
    	"text":"Are you sure you want to do this ?",
    	"ellipses":false,
    	"fontSize":10    	
	});
    $("#gcOpen").click(function(event){
    	event.preventDefault();
    	$("#example2").gatedControl("open");
    });
    $("#gcClose").click(function(event){
    	event.preventDefault();
    	$("#example2").gatedControl("close");
    });    
});	    				
<input type="text" id="example2"/> 
<a href="#" id="gcOpen" >Open</a>
<a href="#" id="gcClose" >Close</a>
Open Close

options:
text : text for the gate.
ellipses : true or false ,default is true. Ellipses is applied for the gate.
fontSize : font-size for the gate.
open : Opens the gate to reveal the control
close : Closes the gate to conceal the control

Callbacks
$(document).ready(function(){
    $('#example3').gatedControl({
        onOpen:function(){
        	alert("opened");
        },
        onClose:function(){
        	alert("closed");	
        }
    });  
});	    				
<input type="text" id="example3"/>

Open Close
Callbacks are:
onOpen : Triggered on open of control
onClose : Triggered on close of control

A more realistic example
